
The file weather2.zip is an exported mule project used as an example with Mulesoft MuleESB and Apache Brooklyn in chapter 3. 

The file "Simple Mule" contains the script to install the server based Mule runtime using Apache Brooklyn in chapter 3. 

The file ifcfg-eth0 is an example ethernet configuration file taken from chapter 3. 

The file "Cassandra Simple Cluster Node" contains the script to install the cloud based Cassandra node using Apache Brooklyn in chapter 3. 

The file "Cassandra Cluster" contains the script to install the cloud based Cassandra cluster using Apache Brooklyn in chapter 3.
